What is JavaScript?

JavaScript is a widely used scripting language on the internet. Recognized and works in all major browsers. Designed to add interactivity and functionality to the Web site.

Types of JavaScript Output

1. window.alert( );

-The alert( ) method displays an alert box with a specified message and an OK button.


2. document.write( );

-The write( ) method writes HTML expressions or JavaScript code to a document.


3. InnerHTML

-The innerHTML property sets or returns the HTML content (inner HTML) of an element.It can be combined with document.getElementById.
